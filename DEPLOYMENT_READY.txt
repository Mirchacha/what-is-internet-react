1. Встановити gh-pages:
   npm install --save-dev gh-pages

2. Оновити package.json:
   Додати: "homepage": "https://YOUR_USERNAME.github.io/what-is-internet-react"

3. Зібрати проект:
   npm run build

4. Задеployити:
   npm run deploy

5. Налаштувати GitHub Pages:
   Settings → Pages → Branch: gh-pages → Save

Готово! 🎉
